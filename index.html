Instructions

  1. Select database
  2. Select Aligner
  3. Select Input Network. To select the network start typing the name of the specie. You can choose Personalized Network in this case you must upload a two column tab separated file with the edges of the network, this proteins must be codified with the selected database
  4. Select Input Network. To select the network start typing the name of the specie. You can choose Personalized Network in this case you must upload a two column tab separated file with the edges of the network, this proteins must be codified with the selected database
  5. In case your database was StringDB you can add threshold to the network, so per example if you select Neighborhood Score and put a value of 600, only edges with neighborhood score higher than 600 will be considered. You can reset the thresholds with the RESET ALL SCORES button
  6. Put your email
  7. Click on send button
  8. When the job is finished you will receive a report with the results
